# SPP 管理員登入流程

## 目的

確保管理員能以正確且安全的方式登入 One Identity Safeguard for Privileged Passwords（SPP）管理介面。

## 適用範圍

- SPP 系統管理員
- 特權帳號稽核人員

## 前置條件

- 已取得 SPP 管理員帳號及初始密碼
- 用戶端裝置可連線至 SPP 管理 IP
- 瀏覽器版本：Chrome 110+ 或 Edge 110+

## 操作步驟

1. 開啟瀏覽器，前往 SPP 管理介面網址（例：`https://<SPP_IP>/`）。
2. 於登入頁面輸入**帳號**與**密碼**。
3. 若環境啟用多因素驗證（MFA），依提示完成第二因素驗證。
4. 登入成功後，畫面將導向 SPP 儀表板（Dashboard）。
5. 確認右上角顯示正確的登入帳號名稱。

## 注意事項

- 密碼連續錯誤超過 5 次將鎖定帳號，請聯繫系統管理員解鎖。
- 請勿使用公用電腦或不受信任的網路環境登入。
- 閒置逾 15 分鐘將自動登出，未儲存的操作將遺失。
